## 3. 功能三：项目列表分组管理

### 3.1 整体布局

```
┌─────────────────────────────────────────────────────────────────────┐
│  🔍 搜索项目...                                    [+ 新建项目]     │
├────────────────┬────────────────────────────────────────────────────┤
│                │                                                    │
│  📁 全部 (12)   │   ┌──────────┐  ┌──────────┐  ┌──────────┐       │
│                │   │ 客户信息  │  │ 订单数据  │  │ 员工名录  │       │
│  📁 工作项目    │   │          │  │          │  │          │       │
│     客户管理(5) │   │ 3 字段   │  │ 5 字段   │  │ 8 字段   │       │
│     订单处理(3) │   │ 128 条   │  │ 1,024 条 │  │ 56 条    │       │
│     数据清洗(2) │   └──────────┘  └──────────┘  └──────────┘       │
│                │                                                    │
│  📁 个人项目    │   ┌──────────┐  ┌──────────┐                     │
│     测试(2)    │   │ 供应商   │  │ 产品库存  │                     │
│                │   │          │  │          │                     │
│  ────────────  │   │ 6 字段   │  │ 4 字段   │                     │
│  + 新建分组    │   │ 89 条    │  │ 234 条   │                     │
│                │   └──────────┘  └──────────┘                     │
│                │                                                    │
└────────────────┴────────────────────────────────────────────────────┘
```

### 3.2 侧边栏分组列表

```
┌────────────────────┐
│ 📋 全部项目 (12)    │  ← 固定项，显示所有项目
├────────────────────┤
│ 📂 工作项目        │  ← 父分组（可折叠）
│    ├─ 客户管理 (5) │  ← 子分组
│    ├─ 订单处理 (3) │
│    └─ 数据清洗 (2) │
├────────────────────┤
│ 📂 个人项目        │
│    └─ 测试 (2)     │
├────────────────────┤
│ ─────────────────  │
│ + 新建分组         │  ← 创建新分组入口
└────────────────────┘
```

### 3.3 分组折叠/展开状态

```
折叠状态:
┌────────────────────┐
│ 📋 全部项目 (12)    │
│ 📁 工作项目 (10)   ▶│  ← 点击展开
│ 📂 个人项目        ▼│  ← 已展开
│    └─ 测试 (2)     │
│ ─────────────────  │
│ + 新建分组         │
└────────────────────┘
```

```

│ ⚠️ 删除分组                                  │

│                                             │

│                                             │
│            [取消]      [确认删除]           │
└─────────────────────────────────────────────┘
```
