## 项目简介

reData 帮助你从格式各异的 Excel 文件中提取结构化数据。只需定义需要的字段，AI 会自动识别表头并完成数据提取。

## 核心功能

- 🎯 **多项目管理** - 项目分组、独立数据隔离
- 📝 **灵活的字段定义** - 像用 Excel 一样定义要提取的字段
- 🤖 **AI 智能识别** - 自动识别表头位置，无需人工标注
- ✅ **数据验证清理** - 自动清理换行、空格等格式问题
- 🔄 **智能去重** - 按指定字段自动过滤重复数据
- 📊 **高级筛选** - 13 种运算符，支持且/或组合筛选
- 📤 **Excel 导出** - xlsx 格式导出，可选字段和范围
- 🔙 **导入撤回** - 以任务为单位撤回整批导入数据

## 快速开始

### 环境要求

- Node.js 18+
- Rust 1.75+

### 安装运行

```bash
cd redata-app
npm install
npm run tauri:dev
```

### 生产构建

```bash
npm run tauri:build
```

## 使用流程

1. **创建项目** - 为你的数据处理任务创建一个新项目
2. **定义字段** - 设置需要提取的字段（姓名、手机号、邮箱等）
3. **配置 AI** - 添加 OpenAI 或其他兼容 API 的密钥
4. **导入文件** - 选择要处理的 Excel 文件
5. **查看结果** - 在结果页面查看、搜索、筛选、导出数据

## 支持的字段类型

| 类型 | 说明 | 示例 |
|------|------|------|
| text | 普通文本 | 备注、描述 |
| name | 姓名 | 2-4个中文字符或英文人名 |
| company | 公司名称 | 含"有限公司"、"集团"等标识 |
| phone | 手机号码 | 自动去除空格和特殊字符 |
| email | 邮箱地址 | 自动转小写 |
| number | 数字 | 年龄、数量 |
| date | 日期 | 支持多种格式 |
| id_card | 身份证号 | 15或18位身份证 |
| address | 地址 | 含省/市/区/路等 |
| url | 网址 | http/https 开头 |

## 导出功能

支持三种导出范围：
- **全部数据** - 所有成功记录（不受筛选影响）
- **当前筛选结果** - 匹配当前筛选条件的数据
- **仅当前页** - 仅导出当前页面的数据

导出为 xlsx 格式，可选包含：
- 自选字段
- 导入时间
- 来源文件
